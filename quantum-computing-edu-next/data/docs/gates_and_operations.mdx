# Quantum Gates and Operations

## Overview

A quantum gate represents a (usually small) operation within a quantum circuit that changes the state of one or more qubits.

In the context of this site, an operation is a specific instance of a quantum gate within a circuit.
An operation includes information such as the qubits acted upon, control qubits, inversion, and repetition.
